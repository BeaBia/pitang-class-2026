# ⚓ Projeto Cais - Terminal de Gestão

O **Cais** é um sistema de gerenciamento de inventário e logística desenvolvido como parte do processo de formação da **Pitang**. O projeto foca em uma experiência de usuário fluida, com uma identidade visual robusta inspirada em terminais portuários.

---

## 🎨 Identidade Visual
O projeto utiliza uma paleta de cores personalizada definida via `oklch` no **Tailwind CSS**:

* **Creme (#F9FBDB):** Fundo e contraste.
* **Teal Principal (#075F5F):** Identidade primária e navegação.
* **Laranja Cais (#B94E27):** Destaques, alertas e ações secundárias.

---

## 🚀 Funcionalidades Principais

* **Autenticação Completa:** Integração com a API **DummyJSON**, com persistência de sessão via cookies e tratamento de erros (401 Unauthorized).
* **Dashboard Inteligente:** Visão geral do sistema com estatísticas em tempo real.
* **Gestão de Inventário:**
    * Listagem de produtos com troca de visualização (**Grid/Tabela**).
    * Paginação dinâmica.
    * **Modal de Detalhes:** Visualização expandida de cada produto sem sair da página.
* **Gestão de Usuários:** Controle e visualização dos operadores do terminal.
* **Interface Responsiva:** Sidebar colapsável e layouts adaptáveis para diferentes telas.

---

## 🛠️ Tecnologias Utilizadas

* **React + Vite:** Base do projeto para alta performance.
* **TanStack Router:** Gerenciamento de rotas tipadas e seguras.
* **TanStack Query:** Cache e sincronização de dados com a API.
* **Tailwind CSS:** Estilização moderna e customização de temas.
* **Shadcn/UI:** Componentes de interface acessíveis e elegantes.
* **Lucide React:** Iconografia temática de logística.
